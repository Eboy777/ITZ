# Норма нагрузки и определение недогруза
## Сделали: Кулик, Олухов, Кияница, Калиниченко

## Задача 11. Норма нагрузки

Норма нагрузки вычисляется по формуле: `830 * ставка`.

- 1 ставка = 830 часов
- 1.5 ставки = 1245 часов (830 × 1.5)
- Дробные ставки рассчитываются пропорционально

## Задача 12. Определение недогруза

Недогруз = нормативная нагрузка - фактически назначенная нагрузка.

Преподаватели с положительным недогрузом участвуют в добалансировке.

## Алгоритм

1. Загрузка листа `19-20` - колонка `Unnamed: 4` - ставка преподавателя
2. Загрузка листа `общее` - колонка `Количества часов по УП` - фактические часы
3. Группировка строк по ФИО с суммированием часов: `df.groupby('ФИО')['Количества часов по УП'].sum()`
4. Для каждого преподавателя вычисляем:
   - норму: `830 * ставка`
   - фактические часы: сумма из листа "общее"
   - недогруз: `норма - фактические`
5. Выводим преподавателей с недогрузом > 0
